🛠️ ドラッグ&ドロップできない時の解決法

3つの簡単な方法で必ず解決します

🎯 解決策1: 個別ファイル選択（推奨）

1. Netlifyの画面で「browse to upload」をクリック
ドラッグ&ドロップエリアの下に小さく表示されているリンクです
2. ファイル選択画面で複数選択
Ctrl+A（Windows）または Cmd+A（Mac）で全選択 または Ctrlキーを押しながら必要なファイルをクリック： - index.html - enhanced-task-manager.html - sales-workflow-builder.html - mobile-app.html - admin-dashboard.html - manifest.json - sw.js
3. 「開く」をクリック
選択したファイルが一度にアップロードされます

💡 解決策2: 圧縮ファイルを作成

1. フォルダを右クリック
「最強タスク管理ツール」フォルダを右クリック
2. 圧縮オプションを選択
Windows: 「送る」→「圧縮（zip形式）フォルダー」
Mac: 「"最強タスク管理ツール"を圧縮」
Linux: 「圧縮」または「アーカイブの作成」
3. ZIPファイルをドラッグ&ドロップ
作成された.zipファイルをNetlifyにドロップ
✅ 既に作成済み: task-manager-deploy.tar.gz
場所: /home/muranaka-tenma/顧客管理ツール/frontend/src/services/api/

🚀 解決策3: GitHub Pages（別の方法）

GitHubで新しいリポジトリ作成
リポジトリ名: sales-task-manager
ファイルをアップロード
GitHubの画面から「uploading an existing file」でファイルを追加
GitHub Pagesを有効化
Settings → Pages → Source: Deploy from a branch → Save
URL: https://[あなたのユーザー名].github.io/sales-task-manager/
⚠️ よくある問題：
• ブラウザのセキュリティ設定でドラッグが制限されている
• フォルダ名に日本語が含まれている
• ファイルサイズが大きすぎる（通常は問題なし）
💡 最も確実な方法：
「browse to upload」から個別ファイルを選択する方法が最も確実です。
フォルダ構造は自動的に保持されます。
🔄 Netlifyに戻って再試行
