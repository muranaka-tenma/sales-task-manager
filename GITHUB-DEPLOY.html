🔗 GitHub → Netlify デプロイ

既存のGitHubアカウントで簡単デプロイ

🚀 方法1: GitHub連携（推奨）

1
新しいリポジトリ作成
GitHubで新しいリポジトリを作成します。
リポジトリ名: sales-task-manager
Public または Private（どちらでもOK）
2
ファイルをアップロード
このフォルダの全ファイルをGitHubにアップロード
# 方法A: GitHubの画面から
「Add file」→「Upload files」→ ドラッグ&ドロップ

# 方法B: Git コマンド
git init
git add .
git commit -m "Initial commit"
git push origin main
3
Netlifyでデプロイ
Netlifyを開く
1. 「Log in」→「GitHub」を選択
2. 「New site from Git」をクリック
3. 「GitHub」を選択
4. 作成したリポジトリを選択
5. 「Deploy site」をクリック

⚡ 方法2: 直接アップロード（超簡単）

1
Netlifyに直接ログイン
Netlify Signup
「Continue with GitHub」を選択
2
フォルダをドロップ
ログイン後、このフォルダ全体をNetlifyにドラッグ&ドロップ
完了！ 即座にURLが生成されます
